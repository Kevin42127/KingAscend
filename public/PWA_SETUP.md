# PWA 圖標設置說明

為了完整支援 PWA，請在 `public` 目錄下添加以下圖標文件：

- `icon-192x192.png` - 192x192 像素的 PNG 圖標
- `icon-512x512.png` - 512x512 像素的 PNG 圖標

圖標設計建議：
- 使用溫暖大地色系（琥珀色/橙色）
- 顯示 "King's Ascend" 或相關王權元素
- 圓角設計，符合應用風格

可以使用在線工具生成：
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

或使用設計工具創建後，將 PNG 文件放置在 public 目錄。
